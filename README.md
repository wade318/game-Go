# 🐣 澂澂的圍棋時光屋 🎯

給低段位小小棋士的 **13 路圍棋** 入門遊戲。純前端、免安裝、免網路，雙擊即玩。

## ✨ 功能

### ⚔️ 對戰
- 雙人對戰，或對「低段位電腦」（啟發式 AI，故意不太強，讓孩子有成就感）
- 悔棋、虛手（Pass）、認輸、重新開始
- **對手等級可調**（入門 20 級 ～ 高手 1 段，共 6 檔）
- **讓子對局**（不讓 / 2 / 4 / 6 / 9 子，讓小朋友先佔星位）
- 提子計數、終局粗略數子判勝負
- **提示一手**：卡住時按一下，用強 AI 建議好點、標在盤上並說明為什麼（純預覽，不會替你落子）
- **即時解說**：每下一手（你的或電腦的）都用白話說明目的 —— 打吃／提子／連接／延伸／自身危險（可開關）
- **吃子小特效**：被提的棋子位置噴彩帶 + 「啵」音效

### 🧩 基礎棋形練習（13 關）
| 關 | 概念 |
|---|---|
| 1 | 認識「氣」 |
| 2 | 打吃（叫吃） |
| 3 | 兩顆一起吃 |
| 4 | 逃跑（長氣） |
| 5 | 連接 |
| 6 | 做出兩隻眼（做活） |
| 7 | 邊上吃子 |
| 8 | 角落抓兩顆 |
| 9 | 一次吃三顆 |
| 10 | 雙打吃（一石二鳥） |
| 11 | 帶著兩顆一起逃 |

- **答對後解說**：說明「這一步的目的」，強化觀念
- **星星獎章 + 進度存檔**（localStorage，關掉重開仍記得）
- 點星星可跳關，過關即得星，全破有盛大慶祝特效

## 🚀 怎麼玩

**最簡單**：直接開 [`澂澂的圍棋時光屋.html`](澂澂的圍棋時光屋.html) 這個單檔（所有程式與圖示都內嵌其中），平板、手機、電腦點開就能玩。

平板加到主畫面：用瀏覽器開啟後 →「分享 → 加入主畫面」，就像一個 App。

### 📖 墨水閱讀器版

給 Kindle / Boox / reMarkable 等 e-ink 裝置的專屬版本：[`澂澂的圍棋時光屋-墨水版.html`](澂澂的圍棋時光屋-墨水版.html)

- 純黑白高對比、實心棋子、無漸層
- **關閉所有動畫、特效、音效**，避免 e-ink 慢刷新造成殘影閃爍
- 落子不做滑鼠預覽（減少重繪）、允許縮放以利閱讀
- 同樣支援對戰與 13 關練習、星星進度

## 🛠 開發（多檔版）

```
index.html          彩色版主頁（對戰 / 練習分頁）
index-eink.html     墨水版主頁（不載入 fx.js）
css/style.css       童趣配色樣式
css/style-eink.css  墨水版灰階高對比樣式
js/engine.js        圍棋規則核心（氣、提子、禁著、打劫、數子）
js/render.js        Canvas 棋盤繪製（支援 { eink: true } 墨水模式）
js/ai.js            低段位電腦對手
js/fx.js            過關 / 吃子特效（墨水版不載入）
js/vs.js            對戰控制
js/practice.js      棋形練習與題庫
icon-180.png        App 圖示
```

改完多檔版後，用以下指令重新打包成單檔：

```bash
python3 build.py
```

> `build.py` 會把 CSS/JS/圖示全部內嵌進 `澂澂的圍棋時光屋.html`。

## 📦 新增練習關卡

編輯 `js/practice.js` 裡的 `PROBLEMS` 陣列，每題格式：

```js
{
  id: 'my-problem',
  title: '第 N 關 · 題目名稱',
  concept: '概念',
  desc: '給小朋友看的說明',
  stones: [ { x, y, color } ],   // color: 1=黑, 2=白
  turn: 1,                        // 輪到誰下（通常黑=玩家）
  solutions: [ { x, y } ],        // 允許的正解落點
  marks: [ { x, y, type: 'circle' } ],
  hint: '點錯時的提示'
}
```

座標為 13 路棋盤、左上角 (0,0)。
